== Description ==

This plugin allows you to disable or enable wordpress users except the admin user.
For more details click [here](http://www.iwebux.com/disableenable-wordpress-users/)

== Installation ==

1. Upload `usercontrol` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Control the user status in User Control admin menu.

== Frequently Asked Questions ==


Does this work for Network/Multisite Wordpress?

Yes.
